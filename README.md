![Orbit1.0](./Orbit1.0.png)

此工具把「Windows工作排程」在建立"定時開啟檔案"的功能獨立出來。
更簡單的直接新增和刪除排程。
同時強化功能，使捷徑檔案(.lnk)也能在指定的時間被啟動
# Day 0 = Everyday

e.g. 每天晚上21:00自動開啟Chrome.lnk、21:01自動開啟line.exe
每週五六日19:31自動開啟MonsterHunterWilds.exe
